# 🏧 키오스크 (Kiosk)

> [← 메인 README로 돌아가기](../README.md)

<br>

## 개요

키오스크는 PC방 내에 설치되는 **무인 자동화 단말기**입니다.
관리자 PC의 TCP 서버(Port 9002)에 연결하여 회원가입, 로그인, 시간 충전, 좌석 확인 기능을 제공합니다.
카운터 직원 없이도 회원이 직접 시간을 충전하고 좌석 현황을 확인할 수 있습니다.

<br>

## 🔄 화면 흐름

```
앱 시작 (Form1 → KioskLogin)
    │
    ▼
[로그인 화면] ── 로그인 성공 ──▶ [충전 화면 (Member_Charge)]
    │                                  │
    │                     ┌────────────┼─────────────┐
    │                     ▼            ▼              ▼
    │               [신용카드]     [카카오페이]   [네이버페이]
    │                     └────────────┴─────────────┘
    │                                  │ 충전 완료
    │                                  ▼
    │                         서버로 CHARGE 전송
    │                         → 관리자 남은시간 실시간 반영
    │
    └──▶ [좌석 확인 화면 (KioskSeat)]
              현재 이용 가능 좌석 시각화 표시
```

<br>

## 🗂 핵심 기능

### 1. 회원 로그인

`KioskLogin.cs` — 관리자 PC 서버(Port 9002)로 로그인을 요청합니다.

**로그인 흐름**
```
사용자 입력 (ID / PW)
    │
    ▼  TCP 송신 (Port 9002)
LOGIN|id|pw  ──▶  서버
    │
    ◀──  LOGIN_OK|이름|남은시간(초)
         또는
         LOGIN_FAIL
    │ 성공
    ▼
Member_Charge 화면으로 전환 (세션 자동 저장)
```

### 2. 시간 충전

`Member_Charge.cs` — 충전권을 선택하고 결제 수단을 고릅니다.

**충전권 종류**

| 금액 | 충전 시간 |
| :---: | :---: |
| 1,000원 | 1시간 |
| 3,000원 | 3시간 30분 |
| 10,000원 | 10시간 |
| 30,000원 | 39시간 |
| 50,000원 | 60시간 |

- 화면에 현재 **아이디·이름·충전시간·남은시간** 표시
- 충전 완료 시 관리자 화면의 해당 좌석 남은시간이 **즉시 갱신**

**충전 통신 흐름**
```
충전권 선택 + 결제 완료
    │
    ▼  TCP 송신 (Port 9002)
CHARGE|id|addSeconds  ──▶  서버
    │
    ◀──  CHARGE_OK|newRemainSeconds
    │
    ▼
키오스크 화면: 남은시간 갱신 표시
관리자 화면:  해당 좌석 remain 라벨 즉시 업데이트
```

### 3. 결제 수단

`Member_Charge.cs` 에서 결제 수단을 선택하면 각 결제 모듈로 연결됩니다.

| 결제 수단 | 파일 | 설명 |
| :---: | :--- | :--- |
| 💳 신용카드 | `Credit.cs` | 카드 결제 처리 |
| 🟡 카카오페이 | `KaKao.cs` | 카카오페이 연동 |
| 🟢 네이버페이 | `Npay.cs` | 네이버페이 연동 |
| 공통 | `Pay.cs` | 결제 공통 로직 처리 |

---

### 4. 좌석 확인

`KioskSeat.cs` — 현재 PC방의 좌석 이용 현황을 시각적으로 보여줍니다.

- 이용 중인 좌석: **파란색** (로그인 PC 강조)
- 비어있는 좌석: **검정색**
- 상단에 이용 가능 좌석 수 표시 (예: `10/50석`)

## 📡 TCP 통신 요약

`KioskTcpClient.cs` — 싱글턴 패턴으로 관리되는 키오스크 전용 TCP 클라이언트입니다.
5초 타임아웃, 응답 대기 중 TaskCompletionSource 패턴을 사용합니다.

| 메시지 | 방향 | 설명 |
| :--- | :---: | :--- |
| `LOGIN\|id\|pw` | →서버 | 로그인 요청 |
| `LOGIN_OK\|name\|remain` | ←서버 | 로그인 성공 응답 |
| `LOGIN_FAIL` | ←서버 | 로그인 실패 응답 |
| `REGISTER\|name\|birth\|id\|pw\|phone` | →서버 | 회원가입 요청 |
| `REGISTER_OK` | ←서버 | 회원가입 성공 |
| `REGISTER_FAIL\|reason` | ←서버 | 회원가입 실패 |
| `CHARGE\|id\|seconds` | →서버 | 시간 충전 요청 |
| `CHARGE_OK\|newRemain` | ←서버 | 충전 완료 응답 |

## 📄 관련 소스 파일

| 파일 | 역할 |
| :--- | :--- |
| `KioskLogin.cs` | 로그인 UI. Port 9002 서버 연결 및 인증 |
| `KioskSeat.cs` | 좌석 현황 시각화 UI |
| `KioskTcpClient.cs` | TCP 클라이언트 싱글턴. 비동기 송수신, 세션 관리 |
| `Member_Charge.cs` | 충전권 선택 및 결제 수단 선택 UI |
| `Credit.cs` | 신용카드 결제 모듈 |
| `KaKao.cs` | 카카오페이 결제 모듈 |
| `Npay.cs` | 네이버페이 결제 모듈 |
| `Pay.cs` | 결제 공통 처리 |
